# Sistema de Inventario con Python y SQLite

Sistema de gestión de inventario desarrollado con Python y SQLite.
Permite registrar productos, controlar stock y consultar inventario.

## 🛠️ Tecnologías
- Python 3
- SQLite3 (base de datos embebida)

## ⚙️ Cómo ejecutarlo

1. Clona el repositorio:
git clone https://github.com/jhenao-dev/inventario-backend.git

2. Entra a la carpeta:
cd inventario-backend

3. Ejecuta el programa:
python3 inventario.pygit add

## 📦 Funcionalidades
- Conexión a base de datos SQLite
- Creación automática de tabla de productos
- Inserción de productos sin duplicados
- Consulta y visualización de inventario
